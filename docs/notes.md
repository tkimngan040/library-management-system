##Database Note
DN1. Database created using Microsoft SQL Server

DN2. Open the Terminal (Ctrl + `) and run "pip install pyodbc" to install the pyodbc library, which is used to connect Python with SQL Server. 

DN3. Open the Terminal (Ctrl + `) and run "python backend/test_db_connect.py" to execute the test_db_connect file.

DN4. Install Flask framework by running "pip install flask" in Terminal.
Flask is used to build the backend web application and APIs.

DN5. Install bcrypt library by running "pip install bcrypt" in Terminal.
bcrypt is used for password hashing and password verification in authentication.

DN6. Install python-dotenv (optional) by running "pip install python-dotenv" in Terminal.
This library is used to load environment variables from a .env file (for security configuration).
